# Bot Browser

The Bot Browser imports character cards from supported card sites without leaving Marinara.

## Supported Sources

Open the **Bot Browser** panel and choose a source such as ChubAI, JannyAI, CharacterTavern, Pygmalion, Wyvern, or DataCat. Availability can change when a site changes its public pages or blocks automated access.

## Import Flow

1. Pick a source.
2. Search by name, tag, or the source's own filters.
3. Open a result to inspect it.
4. Import the card as JSON or PNG when the source provides one.
5. Review the imported character in the Characters panel before using it in a chat.

## Login And Cloudflare Notes

Some sources require you to log in before NSFW or private results appear. If a source is protected by Cloudflare, visit that site once in the same browser, clear the challenge, then retry the search in Marinara.

If a source keeps failing, use the site's own download button and import the character through the normal character import flow.
